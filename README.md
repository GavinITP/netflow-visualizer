# Netflow Visualizer

## Prerequisites

- git
- node.js
- pnpm
- go

## Set up backend

1. Clone the Repository

```
git clone
```

2. Navigate to the backend directory

```
cd backend
```

3. Create .env file

4. Install dependencies

```
go mod tidy
```

5. Start the backend server

```
go run main.go
```

## Set up frontend

1. Install dependencies

```
pnpm i
```

2. Start the frontend

```
pnpm run dev
```
